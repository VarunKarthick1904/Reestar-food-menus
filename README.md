Reestar Menu
This is a simple restaurant menu webpage built using HTML and CSS.
It showcases food items like Chicken Biryani, Pepper Chicken, Egg Gravy, and Parota with descriptions, order buttons, and a contact form.

🔹 Features
   • Single-page layout (index.html + style.css)
   • Menu items arranged using display: inline-block
   • Contact section built with form + table structure
   • Hover effects for buttons and descriptions
   • Clean and beginner-friendly design

🔹 Built With
    • HTML5
    • CSS3
