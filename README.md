Reestar Food Menu – Single Page (HTML & CSS)

This project is a single-page food menu layout created using HTML and basic CSS.
It showcases a simple restaurant-style menu with food images, item names, short descriptions, and an order button — all displayed on one clean page without navigation links.

🚀 Features

• Single-page layout

• Clean and minimal UI

• Food items arranged neatly using basic CSS

• Beginner-friendly code

• Simple and easy to understand structure

🎯 Purpose

• This task helped me practice

• Building a structured single-page design

• Using HTML to display images, text, and buttons

• Applying basic CSS properties like spacing, alignment, and layout
