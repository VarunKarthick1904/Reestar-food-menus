# Reestar Food Menu  🍽️

This is a simple **single-page food menu website** created using **HTML** and **Basic CSS**.  
The page displays different food items with images, descriptions, and a button option to order.  
This project helped me practice layout design, typography, spacing, and basic styling concepts.

## 🛠 Technologies Used
- HTML5
- CSS3

## ✨ Features
- Single page layout
- Food items displayed with image and content
- Clean and simple UI design
- Order button for each item

## 🎯 Purpose
To improve my front-end fundamentals and learn how to structure a page using HTML and style it using CSS.
